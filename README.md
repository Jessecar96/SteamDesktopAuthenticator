# SteamDesktopAuthenticator
Steam桌面令牌工具<br>
工具基于原作者Jessecar96的项目SteamDesktopAuthenticator开发，在原工具上增加令牌转移功能，支持无15天交易限制转移令牌，令牌转移后你在48小时内产生的交易报价将被Steam暂挂，但是你可以正常交易，不会触发15天交易限制<br>
